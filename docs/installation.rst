============
Installation
============


Stable release
--------------

To install JsonStreamReader, run this command in your terminal:

.. code-block:: console

    $ pip install json_stream_reader

This is the preferred method to install JsonStreamReader, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for JsonStreamReader can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/matt2000/json_stream_reader

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/matt2000/json_stream_reader/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/matt2000/json_stream_reader
.. _tarball: https://github.com/matt2000/json_stream_reader/tarball/master
